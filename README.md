## Installation
## 1. Before installing the application, you'll need to obtain google Books API key:
    1. Enable the Books API:
      Go to the Google Cloud Console: https://console.cloud.google.com/
      Create a new project or select an existing one from the project list.
      In the navigation menu, click "APIs & Services" > "Library".
      Search for "Books API" and click "Enable".
    2. Create credentials:
      Go to "APIs & Services" > "Credentials"
      Click "Create Credentials" > "API key"
      You will use this API key when you create a .env file

## 2. Clone the repository using GIT
    git clone https://github.com/Jaine-UB/SkyBridge-Learning.git

## 3. Navigate to the project directory
   cd SkyBridge-Learning

## 4. Install dependencies
npm install

## 5. Create environment variables file
Create a file named .env in the project root directory with the following content.

Google Books API = your_googlebooks_api_key_here

## 6. Tech Stack
- JavaScript
- Node.js
- CSS
- HTML
  
## 7. Start the server
npm run dev
